# LiteUtools

LiteUtools 是一个跨平台桌面工具合集，目前提供两个板块：

- `PDF / 图片转换`
- `文本对比`

## 当前功能

### PDF / 图片转换

- 导入 `PDF`，按页拆分为图片
- 导入多张图片，按当前顺序合并为 `PDF`
- 左侧输入区可移除素材
- 图片素材支持上移、下移调整顺序
- 右侧输出区支持列表 / 缩略图查看
- 图片结果支持软件内预览
- 结果可复制、导出到本地

### 文本对比

- 左右输入两段文本
- 按行查看差异
- 支持长文本独立滚动查看
- 可随时清屏重新对比

## 下载与安装

每次发布新版本后，安装包会在 GitHub Release 中提供：

- macOS: `.dmg`
- Windows: `.exe`
- Linux: `.deb`

## macOS 测试版安装说明

当前 macOS 版本可能是未签名测试包。第一次打开时，如果系统提示无法直接打开，可以按下面的方式处理：

1. 下载并打开 `.dmg`
2. 将 `LiteUtools.app` 拖入 `Applications`
3. 不要直接双击，先在 `Applications` 里对 `LiteUtools.app` 点击右键
4. 选择 `打开`
5. 如果系统继续提示风险，再进入：
   `系统设置 -> 隐私与安全性`
6. 在页面下方找到关于 `LiteUtools` 的拦截提示
7. 点击 `仍要打开`

如果你已经尝试打开过一次，但没有看到允许按钮，可以先回到应用程序目录再次右键 `打开`，然后再去隐私与安全性页面查看。

## 使用方法

### PDF / 图片转换

1. 进入 `PDF / 图片转换`
2. 把 PDF 或图片拖入左侧输入区，或点击按钮选择文件
3. 如果导入的是图片，可以在左侧用 `上移` / `下移` 调整顺序
4. 点击对应的处理按钮开始转换
5. 在右侧输出区查看结果
6. 按需复制结果，或保存到本地

### 文本对比

1. 进入 `文本对比`
2. 在左右两个输入区域分别粘贴文本
3. 中间区域会自动显示差异结果
4. 需要重新开始时点击 `清屏`

## 说明

- 输出区内容属于当前会话的临时缓冲结果
- 关闭软件后，这些临时结果会自动清理
- 重新启动软件后，输出区会重新开始
